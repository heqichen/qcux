# QCUX 使用文档

## 概述

QCUX 是一个 UX 线框图设计工具，用于快速创建网页/应用的交互原型。你可以设计多个界面、添加 UI 元素，并定义界面之间的跳转关系，最后导出为可交互的网页。

## 界面介绍

工具包含两个主界面：

### 交互设计界面（默认界面）

用于管理所有界面、设置界面之间的跳转关系。

**Toolbar（顶部工具栏）：**
| 按钮 | 功能 |
|------|------|
| 📂 打开项目 | 打开 .json 项目文件 |
| 💾 保存项目 | 保存当前项目 |
| 📤 导出 | 导出为独立 HTML 网页 |
| 🔍 显示全局 | 将所有界面最大化居中显示 |
| ➕ 新增界面 | 创建新界面（弹出对话框） |
| 🗑 删除界面 | 删除选中的界面（需先选中） |
| 🏠 设置 Landing Page | 将选中界面设为入口页面 |

**Canvas（画布）：**
- 显示所有界面的缩略图
- 缩略图上方显示界面标题（固定字体大小，不受缩放影响）
- Landing Page 在左下角外侧标记 🏠
- 红色折线箭头表示界面跳转关系
- 点击界面选中，点击空白取消选中
- 拖拽界面可移动位置
- 滚轮缩放（向上放大，向下缩小）
- 鼠标中键拖拽平移
- 双击界面进入界面设计界面

**Toolbox（右侧面板）：**
- 选中界面时：显示界面信息（标题、尺寸、元素数、链接数）、显示"添加交互"按钮
- 选中链接线时：显示链接属性（来源、目标）、删除链接按钮
- 未选中时：提示"选择一个界面或交互链接"

---

### 界面设计界面

用于编辑单个界面的 UI 元素。

**Toolbar（顶部工具栏）：**
| 按钮 | 功能 |
|------|------|
| ➕ 添加物件 | 添加 Object 元素（带对角线的方框） |
| 🔤 添加文字 | 添加 Text 元素 |
| 🔘 添加按钮 | 添加 Button 元素（带下划线文字） |
| ↩ 退出 | 返回交互设计界面 |

**Canvas（画布）：**
- 显示界面区域（白色背景）
- 元素按 z-index 渲染（后创建的在上层）
- 选中元素时显示 8 个拖拽手柄
- 拖拽元素可移动，拖拽手柄可调整尺寸
- 鼠标悬停手柄时显示对应光标图标（移动/缩放方向）
- 滚轮缩放、中键平移

**Toolbox（右侧属性面板）：**
- 选中 Object 时：编辑 X、Y、宽、高、名称
- 选中 Text 时：编辑 X、Y、宽、高、文字内容、字号（A-/A+）
- 选中 Button 时：编辑 X、Y、宽、高、文字内容、字号（A-/A+）

## 操作流程

### 1. 创建界面

1. 点击工具栏「➕ 新增界面」
2. 输入界面标题、宽度（默认 1920）、高度（默认 1080）
3. 点击「确定」进入界面设计界面

### 2. 设计界面

1. 使用工具栏按钮添加 Object、Text、Button 元素
2. 选中元素后在右侧面板调整属性
3. 拖拽元素或手柄调整位置和大小
4. 点击「↩ 退出」返回交互设计界面

### 3. 设置交互跳转

1. 在交互设计界面选中一个源界面
2. 在右侧面板点击「🔗 添加交互」
3. 在弹出的列表中选择触发跳转的元素
4. 鼠标会出现箭头，点击目标界面完成链接
5. 红色折线箭头表示跳转关系

### 4. 管理链接

- 点击红色链接线可选中，右侧面板显示链接详情
- 可查看"来自"哪个界面的哪个元素，以及"去往"哪个界面
- 可点击「🗑 删除链接」移除跳转关系

### 5. 设置 Landing Page

- 默认第一个创建的界面是 Landing Page
- 选中任意界面，点击「🏠 设置 Landing Page」可将它设为入口

### 6. 导出网页

- 点击「📤 导出」生成独立 HTML 文件
- 用浏览器打开，显示 Landing Page
- 点击有交互的元素可跳转到对应界面

## 快捷键与鼠标操作

| 操作 | 方式 |
|------|------|
| 缩放画布 | 滚轮（上滚放大，下滚缩小） |
| 平移画布 | 按住鼠标中键拖拽 |
| 选中/取消 | 点击元素或空白处 |
| 移动元素/界面 | 鼠标左键拖拽 |
| 调整元素尺寸 | 拖拽选中元素的 8 个手柄 |
| 进入界面设计 | 双击界面缩略图 |

---

## 文件格式

项目保存为 JSON 文件，包含：
- `pages`：界面列表（含基本属性和元素列表）
- `links`：跳转关系列表（含来源元素和目标界面）

所有界面尺寸默认为 1920×1080 像素。
